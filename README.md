
# E-Commerce MERN App

Full-stack e-commerce platform built with MERN stack.

## Tech Stack
- Frontend: React.js, Redux
- Backend: Node.js, Express.js
- Database: MongoDB
- Features: JWT Auth, Cart, Admin Dashboard, PayPal Payment
- ## How It Works
- Users can register/login with JWT authentication
- Browse products by category and search
- Add products to cart and place orders
- Pay via PayPal payment gateway
- Admin can manage products, orders and users
- Real-time chat support between admin and users
- Seller dashboard for managing own products
